# /jira-sync — Sync Plan to Jira

Sync the current plan to Jira: $ARGUMENTS

## Instructions

1. Read the plan from the provided path or find the latest in `artifacts/plans/`
2. Confirm the plan status is "Approved"
3. Use the jira-sync agent to create Jira stories from plan phases
4. Report the mapping of plan phases to Jira issue keys
