# /threat-model — Threat Modeling

Perform threat modeling for: $ARGUMENTS

## Instructions

1. Identify assets, trust boundaries, and entry points
2. Apply STRIDE analysis (Spoofing, Tampering, Repudiation, Information Disclosure, Denial of Service, Elevation of Privilege)
3. Score threats using DREAD (Damage, Reproducibility, Exploitability, Affected Users, Discoverability)
4. Recommend mitigations prioritized by risk score
5. Generate a threat model document with diagrams
6. Output to `artifacts/security/`

Use the threat-modeler agent with security-review skill.
