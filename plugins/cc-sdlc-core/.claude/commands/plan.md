# /plan — Multi-Phase Planning

Create a structured implementation plan for: $ARGUMENTS

## Instructions

1. Understand the objective, constraints, and success criteria
2. Explore the codebase for relevant context
3. Decompose into ordered phases with clear deliverables and acceptance criteria
4. Identify risks, open questions, and dependencies
5. Recommend model tiers for each phase (heavy/default/fast)
6. Output using the template at `docs/templates/plan.md`

Use the planner agent and plan-workflow skill.
