# /architect — Architecture Design

Design the architecture for: $ARGUMENTS

## Instructions

1. Understand the problem space, constraints, and quality attributes
2. Explore existing codebase for patterns, dependencies, and tech debt
3. Produce 2-3 architectural approaches with trade-off analysis
4. Recommend an approach and document the decision in ADR format
5. Deliver: component diagram, data flow, migration path (if applicable)
6. Output as Architectural Decision Record to `artifacts/decisions/`

Use the architect agent with plan-workflow skill.
