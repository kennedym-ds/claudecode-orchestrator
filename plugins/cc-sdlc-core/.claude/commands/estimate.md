# /estimate — Effort Estimation

Estimate effort for: $ARGUMENTS

## Instructions

1. Break the work into discrete tasks
2. Assess each task using T-shirt sizing (XS / S / M / L / XL)
3. Map sizes to story point ranges (XS=1, S=2, M=3-5, L=8, XL=13)
4. Apply confidence rating (High / Medium / Low) based on uncertainty
5. Identify risks that could inflate estimates
6. Surface assumptions — ask for clarification if scope is ambiguous

Use the estimator agent.
