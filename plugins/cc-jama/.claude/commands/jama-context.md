# Jama Context

Pull Jama item details into the current session context.

## Usage

```
/jama-context
```

## Workflow

1. Ask for the Jama project and items of interest
2. Retrieve item details with full fields
3. Optionally trace relationships for context
4. Summarize findings in structured format for use in planning or implementation

## CLI Example

```bash
claude "/jama-context"
```
