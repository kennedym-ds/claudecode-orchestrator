## Can't Find Your Role?

- If you write code → [Developer](developer.md)
- If you review others' code → [Tech Lead](tech-lead.md)
- If you define requirements → [Product Owner](product-owner.md)
- If you verify quality → [QA Engineer](qa-engineer.md)
- If you manage releases → [DevOps](devops-engineer.md)
- If you assess risk → [Security Engineer](security-engineer.md)
